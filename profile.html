Description

This Specification is a Profile of PROV-O, the Provenance Ontology, designed to specify the outputs of ProvWorkflow tooling.

The intended use of this Specification is for application code in programming languages such as Python to be written that enables executable workflows to report their actions in a PROV-O compatable way.

The additions to PROV-O made by this Specification's data model (its ontology) are designed to ensure the creation of well connected and well described PROV-O elements and also PROV-O element paterns that work well in data interoggation tooling, such as the SURROUND Ontology Platform.
